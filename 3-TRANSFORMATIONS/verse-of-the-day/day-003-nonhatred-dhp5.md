# Day 3 — Non-hatred (Dhp 5)

Rail: [[2-RAILS/Verses/dhp-5]] · grounded in Sujato (CC0).
Meaning targeted: *hatred is never laid to rest by hatred; only by non-hatred (love). This is an ancient, timeless principle.*
Flagged rendering choice (from the rail): *averena* = "by love" / "by kindness" / "by non-hatred." This card uses **love** for warmth/modern register; the literal "non-hatred" sense is preserved in the termbase row (*avera*).

- **en** — Hatred is never ended by hatred, but only by love. This is an ancient and timeless truth.
