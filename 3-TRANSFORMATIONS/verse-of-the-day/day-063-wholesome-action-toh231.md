# Day 63 — Wholesome action, done with reverence (The Jewel Cloud / Ratnamegha, Toh 231)

Rail: [[2-RAILS/Verses/toh231-merit-with-respect]] · **own translation from the Degé Kangyur (Public Domain)**; 84000 English as reference only (CC BY-NC-ND). A **Tibetan Kangyur Mahāyāna sūtra**.
Short verse, **quoted in full**. Meaning: pursue merit, discipline, and wisdom wholeheartedly, with reverence rather than by rote.

## Source (Tibetan, Toh 231 — verbatim)

> དགེ་སློང་རྣམས་བསོད་ནམས་དག་གུས་པར་བྱའོ། །ཚུལ་ཁྲིམས་གུས་པར་བསྲུང་ངོ་། །ཤེས་རབ་གུས་པར་བསྒོམ་མོ། (`bo-toh231.md#^p32`)
